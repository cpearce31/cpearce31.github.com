Hello!

I'm Caleb Pearce, a self-taught front-end web developer.

My path to being a developer started with completing FreeCodeCamp's front-end course, a comprehensive 400-hour introduction to HTML, CSS, and Javascript. Next, I worked on gaining a deeper understanding of Javascript and broader programming concepts by building things that interested me.

Along the way, I gained a solid understanding of the Linux command line, Git, IDEs, and the nitty-gritty of the web development workflow. Recently, I taught myself React.js and built several projects that make use of it. There's lots more to learn, and that's pretty exciting!

React Game-of-Life

A React.js implementation of Conway's Game of Life, a cellular automaton that simulates the birth and death of single celled organisms.

Interactive Cocktail Guide

An interactive guide to classic cocktails, made with React. Features an autocomplete search and the option to find out what drinks you can make based on what ingredients you have on hand.

Javascript Calculator

A calculator capable of evaluating multi-step expressions. Uses Javascript for expression parsing and jQuery for DOM manipulation.

Noughts and Crosses

A simple Tic Tac Toe game that allows you to play against a friend or the computer.

Responsive Slideshow

A modular version of the slide carousel seen here, ready for use on your website. Features optional autoscrolling and full responsiveness. Made from scratch with plain Javascript.

Opening Lines of Literature

Gets opening lines of famous novels from the WikiQuote API with an AJAX request, then displays a random quote.

Wikipedia Search Viewer

Replicates Wikipedia's search functionality, made to practice getting and displaying data from an API.

Pomodoro Timer

A timer for the Pomodoro time management technique, allowing you to adjust the length of work and rest periods.
